本文展示了一些使用了云湖 API 的开源项目，如果你也想将自己的项目加入以下列表，欢迎对 [本仓库](https://github.com/yh-Tpdev/yhchatAPI) 提出 Pull Request。

## 应用

这些是使用云湖 API 制作的应用，提供了与云湖平台交互的能力，也可称为第三方客户端：

### TypeScript

- [NanoYunHu](https://github.com/TASA-Ed/nanoyunhu): 纯协议无头云湖实现（AGPL-3.0）

### Kotlin

- [Murexide](https://github.com/murexide-project/Murexide): 一个云湖第三方客户端，采用 Material 3 设计。（GPL-3.0）
- [Yhchat_MD3](https://github.com/Kauid323/Yhchat_MD3): 一个云湖第三方客户端，由ai强力驱动编写。（GPL-3.0）

### Java

- [YunHuGram](https://github.com/Kauid323/YunHuGram): 基于 Telegram 的云湖第三方客户端（GPL-2.0）

## 类型定义包

这些是云湖 API 的类型定义包，可以在项目中直接引用。虽然没有直接提供实现或对接功能，但可以帮助开发者快速编写使用云湖 API 的应用或适配器：

### TypeScript

- [yunhu-protobuf-typeproto](https://github.com/ccd2s/yunhu-protobuf-typeproto): 云湖 API 的代码优先 TypeScript Protobuf 类型定义（使用 Typeproto）（AGPL-3.0）

## 适配器

这些是使用云湖 API 制作的适配器，可以用于接入其他的机器人框架。

### Python

- [ErisPulse-YunhuUserAdapter](https://github.com/wsu2059q/ErisPulse-YunhuUserAdapter): 基于云湖用户账户而实现的 ErisPulse 适配器（GPL-3.0）
